HTML

HTML stands for HyperText Markup Language.

It is used to structure web content.

HTML consists of elements used to enclose different parts of content.

Each element has opening and closing tags.

Tags can have attributes to define additional information.

<div>Content</div> <a href="url">Link</a>
  • Headings
  • Paragraphs
  • Links
CSS

CSS stands for Cascading Style Sheets.

It is used to style HTML elements.

CSS can be written inline, internally, or externally.

Selectors target specific HTML elements.

Properties define the styles applied to the elements.

body { background-color: white; } p { color: black; }
  • Classes
  • IDs
JavaScript

JavaScript is a scripting language for the web.

It enables interactive web pages.

JavaScript can manipulate the DOM.

It supports event-driven programming.

Variables can be defined using var, let, or const.

let x = 10; document.getElementById("demo").innerText = x;
  • Events
  • Functions
React

React is a JavaScript library for building UIs.

It uses components to create reusable UI elements.

JSX is a syntax extension used with React.

React uses a virtual DOM for performance.

Hooks are used for managing state and side effects.

function App() { return <h1>Hello</h1>; } useEffect(() => {}, []);
  • Components
  • Props
APIs

API stands for Application Programming Interface.

It allows software systems to communicate.

RESTful APIs are commonly used in web apps.

JSON is a common data format used with APIs.

APIs can be public or private.

fetch('https://api.example.com/data') .then(response => response.json())
  • GET
  • POST
